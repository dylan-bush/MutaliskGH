Grasshopper Plugin Documentation

MutaliskGH

A compiled Grasshopper plugin for migrating a Notion-documented utility library into a maintainable C# codebase with reusable core logic, tests, and family-based component slices.

Current Scope

  • Text, Data, Format, Display, and Geometry slices implemented
  • Pure logic in MutaliskGH.Core
  • Fast regression coverage in MutaliskGH.Tests

Behavior

Current Grasshopper Notes

Display Preview

Color by Branch and Preview Color by Value both generate colors and also preview compatible geometry using those colors directly in the viewport.

Grouped Value Preview

Preview Color by Value filters false-like values, groups retained geometry by distinct value, outputs branched geometry and value trees, and emits one color per branch.

Seed Naming

Both display components use S as the seed input so their IO conventions stay aligned.

ZUI Lanes

RegEx Cull, Branch by Member, and Cull ENF support zoomable || lane expansion for parallel streams.

Formatting Controls

Find Next Available Code supports patterns such as {000###}, and Round Points uses Factor (F) for coordinate rounding.

Geometry Modes

Oriented Bounding Box exposes clustered, mean, and length-weighted direction strategies through Method (M).

Rectangle Ordering

Rebuild Rectangle preserves source geometry and returns vertices ordered bottom-left, top-left, top-right, bottom-right.

Workflow Boundary

Revit automation and other host-side-effect-heavy tools remain intentionally outside the current compiled slice.

Structure

Shared Project Layout

MutaliskGH

Grasshopper wrappers, framework classes, component UI, and viewport preview behavior.

MutaliskGH.Core

Pure reusable logic for text, data-tree manipulation, formatting, display color generation, and geometry helpers.

MutaliskGH.Tests

Fast logic tests used to validate behavior before interactive Grasshopper checks.
