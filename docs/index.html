Grasshopper Plugin Documentation

MutaliskGH

MutaliskGH is a compiled Grasshopper plugin for migrating a curated Notion-documented library of utility components, tree tools, text tools, and workflow helpers into a maintainable C# plugin.

Implemented Components Migration Architecture

Current Scope

The repo currently contains the shared plugin framework, a pure-logic core library, automated tests, and implemented Mutalisk / Text, Mutalisk / Data, Mutalisk / Format, and the first Mutalisk / Geometry slice.

Build

dotnet build MutaliskGH\MutaliskGH.csproj -f net48

Test

dotnet test MutaliskGH.Tests\MutaliskGH.Tests.csproj

Framework

Shared Project Structure

MutaliskGH

Grasshopper-facing wrappers, framework classes, categories, icons, and UI behavior.

MutaliskGH.Core

Pure reusable logic for regex, duplicate analysis, tree grouping, partition rules, formatting tools, and geometry helpers.

MutaliskGH.Tests

Fast non-Grasshopper tests that validate core behaviors before interactive plugin checks.

Interaction Notes

Current Grasshopper Behaviors

ZUI Lanes

RegEx Cull, Branch by Member, and Cull ENF support zoomable || lane expansion for parallel streams.

Pattern Handling

Partition Branches accepts both flat and grafted pattern inputs as long as they represent one branch decision per source branch.

Code Formats

Find Next Available Code supports fixed and searchable slots, including {000###} for the original behavior and literal wrappers such as LVL-{000###}.

Geometry Modes

Oriented Bounding Box supports clustered, mean, and length-weighted edge-direction strategies through Method (M).

Ordered Rectangle Output

Rebuild Rectangle preserves source geometry and returns vertices starting at bottom-left, then clockwise.

Offset Selection

Offset Select returns the selected offset result through Selected Offset (O), or both in ordered output when requested.

Migration Strategy

Revit automation and workflow-heavy tools remain intentionally outside the current compiled slice.
