Grasshopper Plugin

MutaliskGH

A focused Grasshopper toolset for high-utility production work across text processing, tree logic, formatting, display, geometry, Rhino interaction, and Rhino.Inside.Revit.

Current Scope

  • Text, Data, Format, Display, Geometry, Rhino, and Revit families
  • Compiled Grasshopper components with shared core logic and tests
  • Viewport-aware display tools and trigger-safe automation helpers
  • Single product surface under the Mutalisk Grasshopper tab

Highlights

What Defines The Toolset

Deterministic Display

PaletteEngine, Color by Branch, and Preview Color by Value share the same seed-driven color logic for stable, repeatable viewport communication.

Parallel Stream Workflows

Components such as RegEx Cull, Branch by Member, and Cull ENF support zoomable || lane expansion for multi-stream use.

Flexible Geometry Helpers

The geometry family includes point rounding, rectangle evaluation, orientation analysis, offset selection, and extension-trim logic tuned for practical modeling workflows.

Rhino + Revit Reach

MutaliskGH includes native Rhino helpers as well as Rhino.Inside.Revit query and automation components for mixed-environment projects.

Trigger-Safe Automation

Side-effect tools such as rename, zoom, and export operations are gated behind explicit trigger inputs so they behave like deliberate actions instead of passive solve-time effects.

Shared Core Logic

Core behavior lives in MutaliskGH.Core, making the plugin easier to test, maintain, and extend without depending on Grasshopper solve state for every change.
