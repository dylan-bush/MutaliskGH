# Mutalisk Icon Style

Mutalisk component icons should read like Grasshopper toolbar icons first and plugin branding second.

## Core Rules

- Canvas: `24x24` PNG with transparent background.
- Composition: one clear centered pictogram.
- Detail: one primary symbol plus at most one supporting accent mark.
- No text, no tiny UI panels, no decorative frame unless the concept requires it.
- Use medium-weight strokes with rounded caps and joins so the icon survives toolbar scale.

## Color

- Core linework: dark neutral grey.
- Accent: one restrained family color.
- `Text` family accent: `zerg` purple.
- Accent should only emphasize the action or focal mark, not fill the whole icon.

## Visual Character

- Clean, geometric, compact.
- Slightly softened corners.
- High contrast on transparent background.
- Avoid diagram-like layouts or miniature screenshots.

## Family Direction

- `Text`: sharp editorial marks, regex/string metaphors.
- `Data`: branching, grouping, or path logic.
- `Geometry`: precise constructive outlines.
- `Rhino`: selection and document interaction cues.
- `Display`: color and preview cues with restrained saturation.

## Regex Escape Pilot

- Dominant gesture: a diagonal backslash.
- Supporting symbol: a regex metacharacter such as `*`.
- The backslash carries the `Text` family purple accent.
- The icon should read as “regex literalize / escape” in under a second.
